# JobPortal â€“ Role-Based Recruitment Management System

JobPortal is a full-stack web application built using Spring Boot, Spring Security,
Thymeleaf, and MySQL.

The application supports three roles:
- USER
- RECRUITER
- ADMIN

Users can browse and apply for jobs, recruiters can post jobs and manage applications,
and admins can manage users and job listings.

---

## Features

User:
- Register and login
- Browse and search jobs
- Apply for jobs
- View applied jobs

Recruiter:
- Recruiter dashboard
- Post new jobs
- View jobs posted by them
- View applications for their jobs

Admin:
- Admin dashboard
- View all users
- Enable / disable users
- View and delete jobs

---

## Tech Stack

- Java
- Spring Boot
- Spring Security
- Thymeleaf
- MySQL
- Hibernate / JPA
- Maven
- HTML, Bootstrap
- Git & GitHub

---

## How to Run Locally

1. Clone the repository
2. Create a MySQL database named jobportal_db
3. Update database credentials in application.properties
4. Run the application
5. Open http://localhost:8080 in browser
